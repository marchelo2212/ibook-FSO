Actividad de aprendizaje

Tras haber revisado el tema, cada estudiante realizará una práctica de laboratorio considerando las siguientes caractarísticas o elementos:

La práctica deberá ser entregada considerando el siguiente formato:

AQUÍ EL FORMATO DE PRÁCTICAS: Descargar el formato, clic aquí

Algunas indicaciones a considerar:

  1. Contenidos a incluir: El organizador gráfico debe abarcar los contenidos revisados en este tema. 
  2. Fuentes de consulta: Se sugiere emplear las fuentes facilitadas y en caso de considerar necesario, puede buscar otras fuentes. 
  3. Estructura y Organización: Organiza los conceptos de manera clara y coherente, utilizando esquemas como diagramas de flujo, diagramas de proceso, o cualquier otro formato que facilite la comprensión visual.
  4. Elementos Gráficos: Emplea colores, iconos y símbolos adecuados para cada concepto o categoría, resaltando las relaciones entre ellos y mejorando la claridad del organizador gráfico.
  5. Referencias y Fuentes: Asegúrate de incluir referencias a las lecturas proporcionadas en el curso para respaldar los conceptos presentados en el organizador gráfico.
  6. Interactividad: Agrega elementos interactivos como enlaces a recursos adicionales o descripciones emergentes para una experiencia de aprendizaje más dinámica.
